# Villes du Monde

Ce projet est un site web sur le thème des villes du monde.  
Il permet de consulter, rechercher et gérer une base de données de villes avec différentes informations (pays, Fondation, 
Population, etc.).

Le projet a été réalisé avec HTML, CSS, JavaScript, PHP et MySQL en utilisant une architecture structurée.

---

## Fonctionnalités du site

Le site contient 4 pages principales :

### Page d’accueil
- Présentation du site  
- Affichage de photos de villes du monde  
- Interface visuelle et immersive  

---

### Page listing
- Affichage de toutes les villes depuis une base de données  
- Données récupérées via MySQL / PHPMyAdmin  
- Informations détaillées(pays, Fondation, Population, etc.)

 

---

### Page recherche
- Recherche d’une ville
- Affichage des résultats correspondants  
- Système de filtrage dynamique  

---

### Page admin
- Interface d’administration sécurisée  
- Formulaire de gestion des données  
- Ajout / modification des informations dans la base de données  

---

## Architecture du projet

### MVC (Model – View – Controller)
Le projet suit l’architecture MVC :
- Model → gestion des données (base de données MySQL)
- View → interface utilisateur (HTML/CSS)
- Controller → logique du site (PHP, traitement des requêtes)

---

### MLD (Modèle Logique de Données)
La base de données a été conçue avec un MLD structuré :
- Tables des villes
- Tables des attractions
